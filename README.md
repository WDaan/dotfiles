# Postinstall script

> script to automate dev env setup

run with:
```
wget https://raw.githubusercontent.com/WDaan/dotfiles/master/scripts/setup.sh
./setup.sh
```  
can install/setup:
- update system
- zsh
- node
- php/composer
- python3
- kubectl && kubectx (with brew)
- rsub
- docker/docker-compose
